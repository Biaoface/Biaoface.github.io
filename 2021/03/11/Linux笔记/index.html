设置IP地址

  • redhat & centos
vi /etc/sysconfig/network-scripts/ifcfg-ens37

IPADDR=192.168.1.10 #IP地址

NETMASK=255.255.255.0 #掩码

GATEWAY=192.168.1.1 #网关地址

ONBOOT=yes # [yes|no]（引导时是否激活设备

BOOTPROTO=static #[none|static|bootp|dhcp]（引导时不使用协议|静态分配|BOOTP协议|DHCP协议）

DNS1=114.114.114.114

DNS2=8.8.8.8

vi /etc/network/interfaces

SSH免密登陆

  • 生成密钥文件及~/.ssh文件夹
ssh-keygen #客户端服务端都要执行
  • 将客户端 ~/.ssh/id_rsa.pub文件的内容拷贝至服务端 ~/.ssh/authorized_keys文件中

第一次拷贝需要新建authorized_keys

  • 修改配置文件
cp /etc/ssh/sshd_config /etc/ssh/sshd_config.bac #备份配置文件按
sed -i 's/PasswordAuthentication yes/PasswordAuthentication no/' /etc/ssh/sshd_config #关闭密码认证
sed -i 's/#PubkeyAuthentication yes/PubkeyAuthentication yes/' /etc/ssh/sshd_config #开启公钥认证
systemctl restart ssh #重启ssh服务

防火墙设置

  • 禁止PING
iptables -A INPUT -p icmp --icmp-type 8 -s 0/0 -j DROP

–icmp-type 8 echo request 表示回显请求

  • 允许PING
iptables -A INPUT -p icmp --icmp-type echo-request -j ACCEPT
iptables -A OUTPUT -p icmp --icmp-type echo-reply -j ACCEPT

增加交换分区

free -h #查看交换分区大小
swapoff -a #删除交换分区
dd if=/dev/zero of=/usr/swap/swapfile bs=1M count=2048 #创建swap分区文件
mkswap /usr/swap/swapfile #格式化交换分区文件
sed -i 's//swap none swap sw 0 0/#swap none swap sw 0 0/' /etc/fstab #注释swap none swap sw 0 0行
echo "/usr/swap/swapfile swap swap defaults 0 0" >> /etc/fstab #添加行

bs 是每块的大小(不超过free -h中buff/cache的值)。

count 是块的数量。

bs*count 为swap文件大小，1M2048=2G。
